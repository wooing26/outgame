Animated Sci-Fi Props Pack - 3D Microgames Add-Ons
========================================

This Microgame Add-On is meant to be used with:
- "FPS Microgame" Version 3.0.0 available from the Unity Hub.
- OR "Karting Microgame" Version 4.0.0 available from the Unity Hub.
- In the Unity Hub, select the "Projects" tab, click the "New Project" button, from the list of Templates, select the "FPS Microgame" or "Karting Microgame".

You can find more information about this Microgame Add-On here: 
http://u3d.as/1PVb
